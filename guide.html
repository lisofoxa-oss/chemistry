А что это вообще такое?

Представь пиццу на двоих
Если один друг «жаднее» — он заберёт больше кусков. Степень окисления — это сколько «электронных кусков» условно забрал или отдал атом.

Степень окисления — это условный заряд атома в молекуле.

Это просто число со знаком: +1, −2, +6, 0... Показывает, «кто кому сколько должен» электронов.

Не бойся — считать это проще, чем кажется!

1
Выучи 5 правил — они решают всё
🏠
Простое вещество = 0
Na, Fe, O₂, Cl₂, H₂ — если элемент «один дома», он никому ничего не отдал.
🔴
Кислород (O) = −2
Почти всегда. Кислород — жадина, забирает 2 электрона у всех.
💧
Водород (H) = +1
С неметаллами (HCl, H₂O). Но с активными металлами (NaH, CaH₂) он = −1.
⚙️
Металлы — фиксированные значения
Ia группа (Na, K, Li) = +1  IIa (Ca, Mg, Ba) = +2  Al = +3
⚖️
Сумма в молекуле = 0
Все плюсы и минусы в молекуле должны уравновеситься до нуля. Это — твой главный инструмент!

Вот и всё! Пять правил. Серьёзно, больше ничего не нужно.

Простое вещество (Na, O₂, Cl₂, Fe) = 0

Элемент сам по себе — «нулевой баланс», никому ничего не должен.

↓
3
Решаем простые — на автомате
Пример 1 — самый простой
+1H₂−2O

Водород = +1, Кислород = −2. Готово!

(+1)×2 + (−2)×1 = +2 − 2 = 0 ✓

Сумма = 0. Сходится!

Пример 2 — ищем хлор
+1Na?Cl
  • Na = +1 (металл Ia группы)
  • Сумма = 0, значит: +1 + Cl = 0
  • Cl = 0 − 1 = −1 ✓
Пример 3 — ищем углерод
?C−2O₂
  • O = −2, атомов кислорода 2
  • C + (−2)×2 = 0
  • C − 4 = 0 → C = +4 ✓

4
Теперь посложнее — но тот же приём!
🧩
Всё как пазл
Знаешь всё кроме одного элемента → составляешь уравнение → находишь неизвестное. Каждый. Раз. Одинаково.
Пример 4 — серная кислота
+1H₂?S−2O₄
  • H = +1, их 2 штуки → (+1)×2 = +2
  • O = −2, их 4 штуки → (−2)×4 = −8
  • S = ? Составляем: +2 + S + (−8) = 0
  • S − 6 = 0 → S = +6 ✓
Пример 5 — перманганат калия
+1K?Mn−2O₄
  • K = +1 (одна штука)
  • O = −2, их 4 → (−2)×4 = −8
  • +1 + Mn + (−8) = 0
  • Mn − 7 = 0 → Mn = +7 ✓
Пример 6 — ловушка!
+2Ca?H₂
  • Стоп! Это гидрид металла — H тут = −1 (не +1!)
  • Ca = +2 (IIa группа)
  • +2 + (−1)×2 = +2 − 2 = 0 ✓

Водород = −1 только с активными металлами!

Пример 7 — два неизвестных атома
+1K₂?Cr₂−2O₇
  • K = +1, их 2 → +2
  • O = −2, их 7 → −14
  • Cr = ?, их 2: +2 + 2×Cr + (−14) = 0
  • 2×Cr = 14 − 2 = 12 → Cr = +6 ✓

Не забывай умножать на количество атомов!
